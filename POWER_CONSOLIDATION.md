### Säule 2: **Konkurrenten eliminieren** (Network Position)

**Problem:** Andere Akteure könnten das Vakuum auch füllen wollen.

**Strategien:**

#### A) Physische Elimination
```yaml
Methode: Töten/Verhaften/Exilieren

Beispiele:
  - Stalin: Große Säuberung (1936-1938)
    → Alle potentiellen Rivalen eliminiert
  - Mao: Kulturrevolution
    → Alte Garde ausgeschaltet
  - Saddam Hussein: Baath-Partei-Säuberung (1979)
    → Rivalen vor laufender Kamera verhaftet

Effektivität: 9/10 (kurzfristig)
Risiko: Erzeugt Widerstand, internationale Ächtung
Legitimacy Impact: -2 (aber wen kümmert's wenn alle Gegner tot sind?)
```

#### B) Ko-optierung
```yaml
Methode: Gegner einbinden, korrumpieren, kaufen

Beispiel: Augustus (Rom, 27 BC)
  Problem: Viele Senatoren könnten rebellieren
  Lösung:
    - Behielt Senat (formal)
    - Gab ihnen Ehrentitel
    - Bezahlte sie gut
    - Ließ sie "wichtig" fühlen
    → Ohne echte Macht

Effektivität: 7/10 (langfristig besser)
Risiko: Gegner könnten sich später wenden
Legitimacy Impact: 0 (neutral)
```

#### C) Institutionelle Neutralisierung
```yaml
Methode: Regeln ändern, Strukturen umbauen

Beispiel: Orbán (Ungarn, 2010+)
  Problem: Opposition, freie Medien, unabhängige Justiz
  Lösung:
    - Verfassung umschreiben
    - Wahlrecht ändern (begünstigt Fidesz)
    - Medien aufkaufen (durch Vertraute)
    - Justiz mit Loyalisten besetzen

  Ergebnis:
    - Opposition existiert noch (formal)
    - Aber strukturell machtlos

Effektivität: 8/10
Risiko: International kritisiert, aber schwer umkehrbar
Legitimacy Impact: -1 (aber legal!)
```

**Problem:** Du hast Macht, aber kontrollierst nicht die Ressourcen.

```python
def secure_resource_control():
```

```yaml

  - Aramco = staatlich
  - König kontrolliert Einnahmen
  - Verteilt Geld an Loyalisten
  - Kauft Stabilität

  Resource Control: 9
  → Macht ist sicher, solange Öl fließt
```

---

#### B) Patronage-Netzwerke
```yaml
Methode: Verteile Ressourcen an Unterstützer

Struktur:
  Level 1: Du (Zentrum)
    ↓ (Geld, Posten, Privilegien)
  Level 2: Innerer Kreis (10-50 Personen)
    ↓ (Kleinere Stücke)
  Level 3: Mittlerer Kreis (500-5000)
    ↓ (Krümel)
  Level 4: Basis (Millionen)

Regel: Jeder Level ist abhängig von deiner Gunst

Beispiel: Russland unter Putin

  - Oligarchen (Wirtschaft)
  - Gouverneure (Regional)

  → Network Position gesichert
```

```python
def survive_long_enough():
```

## Die ultimative Konsolidierungs-Checkliste

```python
def consolidate_power(initial_power):
    """7-Säulen-Konsolidierung"""

    checklistcompleted = {
        'legitimacy_built': False,
        'competitors_eliminated': False,
        'resources_secured': False,
        'information_controlled': False,
        'violence_monopolized': False,
        'bureaucracy_established': False,
        'time_survived': 0
    }

    while not all_consolidated():

        # Säule 1: Legitimität
        if not checklist['legitimacy_built']:
            choose([
                deliver_performance(),
                craft_narrative(),
                hold_elections()
            ])
            if legitimacy > 6:
                checklist['legitimacy_built'] = True

        # Säule 2: Konkurrenten
        if not checklist['competitors_eliminated']:
            for rival in rivals:
                choose([
                    eliminate(),
                    co_opt(),
                    neutralize()
                ])
            checklist['competitors_eliminated'] = True

        # Säule 3: Ressourcen
        if not checklist['resources_secured']:
            control_economy()
            build_patronage()
            checklist['resources_secured'] = True

        # Säule 4: Information
        if not checklist['information_controlled']:
            control_media()
            shape_education()
            checklist['information_controlled'] = True

        # Säule 5: Gewalt
        if not checklist['violence_monopolized']:
            secure_military()
            disarm_opposition()
            checklist['violence_monopolized'] = True

        # Säule 6: Bürokratie
        if not checklist['bureaucracy_established']:
            build_institutions()
            professionalize()
            checklist['bureaucracy_established'] = True

        # Säule 7: Zeit
        wait(1_year)
        checklist['time_survived'] += 1

        if checklist['time_survived'] > 5:
            legitimacy += 1  # Zeit legitimiert

    return "POWER CONSOLIDATED"
```

### Die 3 Wege der Konsolidierung

1. **Autoritär (schnell, 1-3 Jahre)**
   - Brutal, effizient
   - Stabil kurzfristig
   - Fragil langfristig (Legitimacy niedrig)

2. **Hybrid (mittel, 5-10 Jahre)**
   - Demokratische Fassade
   - Autoritäre Realität
   - Moderat stabil

3. **Demokratisch (langsam, 15-30 Jahre)**
   - Konsens, Legitimität
   - Fragil kurzfristig
   - Sehr stabil langfristig
