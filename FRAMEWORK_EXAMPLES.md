## Praktische Evaluations-Checkliste

### Schritt 1: Daten sammeln

Für jede Dimension, identifiziere:
- [ ] Konkrete Fähigkeiten
- [ ] Messbare Metriken
- [ ] Limitierungen
- [ ] Rechtliche/ethische Grenzen

### Schritt 2: Sub-Dimensionen bewerten

Nutze diese Skala für jede der 30 Sub-Dimensionen:

```
0 = Keine Fähigkeit
1 = Persönliche Ebene
2 = Lokale Ebene
3 = Kleine Organisation
4 = Regionale Ebene
5 = Staatsebene / Spezialisiert
6 = Nationale Ebene / Domain-Expert
7 = Multi-National / Multi-Domain
8 = Global / Cross-Institutional
9 = Universal / Nahezu perfekt
```

### Schritt 3: Berechne Scores

```python
# Für jede Dimension
dimension_score = sum(sub_dimensions) / 6

# Gesamtscore
total_power = sum(all_dimensions) / 5

# Balance
import statistics
balance_factor = 1 - (statistics.stdev(dimensions) / statistics.mean(dimensions))
```

### Schritt 4: Interpretiere

- **Total Power < 3:** Limitiertes System
- **Total Power 3-5:** Operational/Tactical
- **Total Power 5-7:** Professional/Academic
- **Total Power 7-9:** Strategic/Global
- **Total Power 9+:** Theoretisch

- **Balance < 0.5:** Sehr unbalanciert, kritische Schwachstellen
- **Balance 0.5-0.7:** Unbalanciert, prüfe schwache Dimensionen
- **Balance 0.7-0.85:** Moderat balanciert
- **Balance > 0.85:** Gut balanciert

### Schritt 5: Identifiziere Wachstumspfade

Für jede niedrig-bewertete Dimension:
1. Was fehlt?
2. Was würde nötig sein, um +1 Level zu erreichen?
3. Ist das realistisch/wünschenswert?
